\doxysubsection{Detailed Description}
\doxylink{namespace_direct_x_helper}{Direct\+XHelper} namespace~\newline
 This namespace contains helper functions for DirectX. It contains functions for creating constant buffer, vertex buffer, index buffer, vertex shader, and pixel shader. It also contains functions for compiling shader and reading shader.
