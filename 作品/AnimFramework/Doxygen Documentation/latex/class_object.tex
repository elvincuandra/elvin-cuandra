\doxysubsection{Detailed Description}
\doxylink{class_object}{Object} class. 

This class is the base class for all objects. It is responsible for the initialization, update, rendering, and termination of components contained within the container, as well as \doxylink{class_object}{Object}\textquotesingle{}s child classes\textquotesingle{}.
