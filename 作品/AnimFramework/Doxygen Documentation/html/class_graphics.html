Detailed Description

Graphics class (Renderer class)

This class is responsible for initializing, updating, drawing, and terminating the graphics. It initializes the graphics, sets the world, view, and projection matrices, and draws the graphics. It is essentially a wrapper for the Direct3D 11 API.
