Detailed Description

FPS class.

This class is responsible for controlling the frame rate of the application. It calculates the time passed since the last frame and waits for the remaining time to meet the desired frame rate. It has two options : VFR and CFR. VFR (Variable Frame Rate) : The frame rate is not fixed. It is calculated dynamically. CFR (Constant Frame Rate) : The frame rate is fixed. It is set by the user. CFR uses thread to wait for the remaining time to meet the desired frame rate.
